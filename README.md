# 💌 Happy Valentine's Theme

A cute, interactive Valentine's Day page — an envelope you click to open a little love letter, and a playful "Would you date me?" card where the **Yes** button grows more tempting while the **No** button shyly runs away across the whole screen. 🥰

## ✨ Features

- **Click-to-open envelope** with a smooth flap + rising letter animation
- A centered wax-seal **heart** that dips when you hover
- A playful proposal card where:
  - the **Yes** button gently grows each time you tease it
  - the **No** button flees to a random spot anywhere on the page (good luck clicking it 😉)
  - the **No** button's message gets cuter every time
- Floating hearts, soft pastel colors, and a "Play Our Song" button
- Responsive layout for desktop and phone

## 🛠️ Built With

- **HTML** — structure
- **CSS** — styling, layout, and all the animations
- **A sprinkle of JavaScript** — for the envelope, the runaway button, and the audio

## 🚀 How to Run

No build tools needed — just open the file in a browser:

1. Download or clone this folder
2. Double-click `index.html` (or right-click → Open With → your browser)
